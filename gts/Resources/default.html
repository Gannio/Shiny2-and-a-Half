This is the default page for Shiny². The host can configure what is displayed here inside the program by modifying the "BrowserMessage" tab.
Here are some basic tips for using Shiny²'s HTML output:
  • I recommend remotely linking to your images rather than using the Base64 embed option. That can really bulk up your page.
  • You have to do everything inline or embedded. You -can- technically link all your code from another website if you want.
Here are the currently supported macros of Shiny². Please note, this list can grow or shrink over time:
  • $hg$ - This will display how many Generation IV Pokémon have been sent.
  • $bw$ - This will display how many Generation V Pokémon have been sent.
  • $mode$ - This will display the current mode that the server is in, whether it is receiving or distributing.
  • $stime$ - Displays a short time string [e.g.; 1:30PM]
  • $ltime$ - Displays a long time string [e.g.; 1:30:44 PM]
  • $sdate$ - Displays a short date string [e.g.; 25.02.2012]
  • $ldate$ - Displays a long date string [e.g.; Saturday, February 25th, 2012]
  • $cgen4$ - Displays the current Generation IV Pokémon being distributed.
  • $cgen5$ - Displays the current Generation V Pokémon being distributed.
  • $fgen4$ - Displays a generated table showing the Generation IV distribution folder.
  • $fgen5$ - Displays a generated table showing the Generation V distribution folder.
